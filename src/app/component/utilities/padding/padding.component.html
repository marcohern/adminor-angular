Media Query Padding

It is Very Easy to Customize and it uses in website apllication.

Class Value
.pd-[size]-t-[value]
.pd-[size]-r-[value]
.pd-[size]-b-[value]
.pd-[size]-l-[value]

size: xs | sm | md | lg | xl

value: the padding value (refer to code above)
